# Security Policy

## Supported Version

The actively maintained version is `1.1.x`.

## Reporting Issues

If you find a security issue, do not publish exploit details publicly. Open a private report through GitHub Security Advisories when available, or contact the repository owner.

## Download Safety

AetherFlow shells out to bundled `yt-dlp` and `ffmpeg` binaries. Only replace these tools with builds from trusted sources.
